# BWDAI_1
Bases for Working with Data for AI project (end of 1st semester)

## Анализ данных Google Books

Учебный проект по анализу датасета книг Google Books с использованием Python и инструментов визуализации.

### Описание проекта

Цель работы — провести полный цикл анализа данных: от первичной загрузки и очистки до построения статистических моделей и визуализации зависимостей.

#### Основные этапы

**Первичный анализ**: Загрузка CSV, проверка структурных типов данных и обзор первых/последних строк.

**Обработка данных**:

Поиск и заполнение пропусков (использование медианы для числовых данных).

Удаление дубликатов.

Поиск выбросов с помощью метрики Z-score.

**Статистика**: Расчет описательных статистик и корреляционной матрицы.

**Визуализация**: Построение гистограмм, линейных графиков, боксплотов и тепловой карты корреляций.

**Агрегация**: Группировка данных по категориям и языкам, создание сводных таблиц.

#### Стек технологий

Python 3.x

Pandas, Numpy — обработка и анализ.

Matplotlib, Seaborn — визуализация.

Scipy — статистические методы.

#### Инструкция по запуску

Убедитесь, что файл google_books_dataset.csv находится в корневой папке проекта.

Установите необходимые зависимости:

pip install pandas numpy matplotlib seaborn scipy


Запустите скрипт анализа:

python google_books_analysis.py
